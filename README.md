# World Timer — Release

Static download / release page for **World Timer**, a desktop app that shows time, weather, news, and sports scores for every city a global team works in.

This repo is intentionally minimal — it exists to host the public download page for built application binaries. The marketing site lives in [`clockwise-landing`](https://github.com/vayuputra/clockwise-landing) and the application source is maintained separately.

## Deployment

Deploy this directory as-is to any static host (Vercel, Netlify, GitHub Pages, Cloudflare Pages, S3, …). No build step is required.
